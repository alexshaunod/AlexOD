CopterKid



Details

    Platform
  • Android
    Release Date
  • 21/08/2017

CopterKid is a side scrolling helicopter game that brings the addictiveness of the classic flash
game to mobile. The game is written in C++ using the Cocos2Dx game engine.
The player avoids obstacles to travel the furthest distance for the high score.
CopterKid is available for free here.
